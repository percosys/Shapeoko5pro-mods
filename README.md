# Shapeoko 5 Pro mods
This repo contains a collection of modifications I have created for the Shapeoko 5 Pro 4x4 CNC machine. 

<img src="./Images/Shapeoko 5 pro - Percosys.PNG" alt="Shapeoko 5 Pro" width="1280" height="640"/>

## Raised gantrys
Raises the gantry by three inches. Particularly useful if you plan on installing an ATC Spindle. 

<img src="./Raised%20Gantry/Images/reverse%20chain%20mount.png" alt="Raised Gantry" width="1280" height="640"/>

## Reverse Drag chaines 
Reposition the drag chains so the cables come out the front of the machine when using the raised gantry mod. This particularly is useful if you are running into clearence issues when using a dust boot. 

<img src="./Images/Reverse%20chains.PNG" alt="[Reverse chains" width="1280" height="640"/>

## CNCDepot Rear mount Tram mount
Mounting plates for both mounting and traming a RM series CNCDepot spindles. It uses a common traming mechanism for cnc machines and provides lower and parallel mounting options. 

<img src="./Images/RM%20mount.png" alt="RM40 mounts" width="1280" height="640"/>

## Pneumatic Dust boot
Dust boot that works with the RM tram mount mod to provide a way to pneumatically control the position. 

<img src="./Images/Pneumatic%20Dust%20Boot.PNG" alt="Dust Boot" width="1280" height="640"/>

## MASSO Touch breakout enclosure
Enclosure that breaks out all the connections needed to convert a Shapeoko 5 pro to MASSO. 

<img src="./Images/Masso%20breakout.PNG" alt="MASSO Breakout" width="1280" height="640"/>

## ATC Tool Rack mounts
Collection of mounts to attach tool holders to the bed of the Shapeoko.

<img src="./Images/Tool%20Holder.PNG" alt="Tool Holder standoffs" width="1280" height="640"/>

## Generic Tool probe mount
Provides a way to attach generic tool probes you typically find on Amazon and Aliexpress. 

<img src="./Images/tool%20probe.PNG" alt="Probe mount" width="1280" height="640"/>

## PSU Enclosure
Enclosure to hold a 24v Meanwell PSU for powering the controller and a 36v Meanwell PSU for powering stepper motors. It also offers a earth ground on the controller connection to ground the chasis. Multiple enclosures can be used if driving high current steppers.

<img src="./Images/psu%20enclosure.PNG" alt="PSU enclosure" width="1280" height="640"/>

## Power contactor enclosure
Allows both 120vac and 240vac power circuits to be controlled remotely. It also provides an E-stop switch so that power can be cut from both circuits at the same time.  

<img src="./Images/contactor%20box.png" alt="Contactor Box" width="1280" height="640"/>
